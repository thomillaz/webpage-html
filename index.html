O que é HTML?

( Voltar* )

HTML é a sigla para Hypertext Markup Language, que em português significa "Linguagem de marcação de HiperTexto". Apesar de algumas pessoas confundirem, HTML não é uma linguagem de programação, mas uma linguagem de marcação, como seu próprio nome sugere. Em outras palavras, essa linguagem não utiliza lógica matemática (cálculos) e nem variáveis em sua estrutura.

Seu papel é identificar as partes de uma página ao navegador, como por exemplo: o que, dentro da página, é o título principal, o que é um link e para onde direciona, o que é o cabeçalho ou rodapé, e assim por diante.

Tags da estrutura principal:

( Voltar* )

As tags principais do HTML, isto é, as que formam a estrutura da página, são:

  • !Doctype html: Indica ao navegador que o arquivo é um documento HTML.

  • html: É uma das tags da linguagem que possui abertura e fechamento, serve para iniciar e fechar o documento HTML.

  • head: Significa "cabeça", e é onde se colocam algumas informações importantes, como o título da página (aquele que aparece na aba do navegador), favicon (o símbolo/logo que identifica uma marca), e por aí vai.

  • body: Significa "corpo" e é onde se posicionam, em excessência, todas as outras tags da página.

      Algumas tags que podem ser utilizadas dentro de body para um código semântico são:

    • header: Significa "cabeçalho", e pode ser utilizada para delimitar o cabeçalho da página. Por exemplo, título, descrição e menu de navegação.
    • main: é utilizada para demarcar a área principal da página.
    • footer: Significa "rodapé" e pode ser utilizado para marcar o rodapé da página, como também de elementos dentro da página, como divs, por exemplo.

Tags do Texto

( Voltar* )

Para estilizar a escrita e tornar o conteúdo mais apresentável e organizado, existem algumas tags que podem auxiliar:

  • h1, h2, h3, h4, h5, h6: São utilizados para demarcar títulos, sendo o h1 o título principal, e os demais, representando títulos menores;
  • p: Demarca parágrafos;
  • br: Representa um espaço, ou quebra de linha;
  • b ou strong: marcam o texto em negrito;
  • i: marca o texto em itálico;
  • strike ou del: risca o texto, pode ser utilizado para dizer que uma informação não é mais válida;
  • blockquote: utilizado para destacar um bloco de texto;
